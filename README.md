Tavern
=============
[![Discord](https://img.shields.io/discord/1480915716226945066.svg?color=%237289da&label=Discord&logo=discord&logoColor=%237289da)](https://discord.gg/R6z5AXmZzp) [![Support](https://img.shields.io/badge/Ko--fi-F16061?style=for-the-badge&logo=ko-fi&logoColor=white)](https://ko-fi.com/mobby_hytale)

Tavern is a Minecraft Dungeons Mod Loader based that support lua, c++ and BluePrint (.pak) mods!

# How to install
- Download the .zip on the [Releases](https://github.com/mobby-hytale/Tavern/releases/tag/Tavern) section of Tavern
- Extract the archive on `/Dungeons/Binaries/Win64`

# How to Build

- `git clone https://github.com/mobby-hytale/Tavern.git`
- `cd Tavern`
- `make` then move the TavernModLoader.dll in the Win64 folder of you game
- `make proxy` then move the xinput1_3.dll (steam) or the xinput1_4.dll (Microsoft store) file in your Win64 folder if you want the Loader to be injected automaticly when the game launch

# How to install Mods
- Place your mod on the `/Dungeons/Binaries/Win64/ue4ss/Mods/` folder
